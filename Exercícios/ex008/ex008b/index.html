Exemplo de código em Phyton

Sem tag pre:

num = int(input('Digite um número')) if num % 2 == 0: print(f'0 número {num} é par') else: print(f'0 número {num} é ímpar') print('FIM DO PROGRAMA')

Com tag pre:

        
            num = int(input('Digite um número'))
            if num % 2 == 0:
                print(f'0 número {num} é par')
            else:
                print(f'0 número {num} é ímpar')
            print('FIM DO PROGRAMA')

Para deixar grudado na tela o código, utilize shift+tab e para voltar para o lado aperte apenas tab, assim:

    
    num = int(input('Digite um número'))
    if num % 2 == 0:
    print(f'0 número {num} é par')
    else:
    print(f'0 número {num} é ímpar')
